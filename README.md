# VueForge

VueForge is a Vue 3 ecosystem for design-system foundations, theming, layouts, icons, and interactive playground experiences.

## Packages

- `@codemonster-ru/vueforge-core` - Core Vue 3 components, tokens, and theme runtime. [npm](https://www.npmjs.com/package/@codemonster-ru/vueforge-core) · [docs](https://docs.codemonster.net/vueforge/core/)
- `@codemonster-ru/vueforge-theme` - Framework-agnostic token and preset engine. [npm](https://www.npmjs.com/package/@codemonster-ru/vueforge-theme) · [docs](https://docs.codemonster.net/vueforge/theme/)
- `@codemonster-ru/vueforge-layouts` - Responsive app-shell and document layout primitives. [npm](https://www.npmjs.com/package/@codemonster-ru/vueforge-layouts) · [docs](https://docs.codemonster.net/vueforge/layouts/)
- `@codemonster-ru/vueforge-icons` - Lightweight Vue 3 icon library with a unified API. [npm](https://www.npmjs.com/package/@codemonster-ru/vueforge-icons) · [docs](https://docs.codemonster.net/vueforge/icons/)
- `@codemonster-ru/vueforge-codeblock` - Vue 3 code block component with syntax highlighting and token-based theming. [npm](https://www.npmjs.com/package/@codemonster-ru/vueforge-codeblock) · [docs](https://docs.codemonster.net/vueforge/codeblock/)
- `@codemonster-ru/vueforge-playground` - Vue 3 UI adapter for interactive playground sessions. [npm](https://www.npmjs.com/package/@codemonster-ru/vueforge-playground) · [docs](https://docs.codemonster.net/vueforge/playground/)
- `@codemonster-ru/vueforge-playground-core` - Framework-agnostic runtime core for playground sessions. [npm](https://www.npmjs.com/package/@codemonster-ru/vueforge-playground-core) · [docs](https://docs.codemonster.net/vueforge/playground-core/)
