## Design Tokens

Available design tokens and corresponding CSS variables.

| Token | CSS Variable | Description |
| --- | --- | --- |
| Header height | `--vf-layout-header-height` | Outer block size of a header area. The default is `4rem`; padding and the bottom border are included in that size. |
| Header block padding | `--vf-layout-header-padding-block` | Vertical spacing inside a header. Set it together with the header height so controls fit without changing the header’s block size. |
| Header inline padding | `--vf-layout-header-padding-inline` | Horizontal spacing inside a header. |
| Header background | `--vf-layout-header-background` | Header surface color. |
| Header border | `--vf-layout-header-border` | Bottom border of a header area. |
