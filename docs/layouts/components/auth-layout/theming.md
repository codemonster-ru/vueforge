# Theming

Component styling surface: available classes and design tokens.

## CSS Classes

Documented CSS classes and their intended usage.

| Class | Description |
| --- | --- |
| `.vf-auth-layout` | Root layout element. |
| `.vf-auth-layout__container` | Width container around the auth panel. |
| `.vf-auth-layout__panel` | Auth content panel. |
| `.vf-auth-layout__header` | Brand, title, and description wrapper. |
| `.vf-auth-layout__brand` | Brand slot wrapper. |
| `.vf-auth-layout__title` | Title element. |
| `.vf-auth-layout__description` | Description element. |
| `.vf-auth-layout__body` | Default slot wrapper. |
| `.vf-auth-layout__footer` | Footer slot wrapper. |

## Design Tokens

Available design tokens and corresponding CSS variables.

| Token | CSS Variable | Description |

| `authLayoutPaddingBlock` | `--vf-layout-auth-layout-padding-block` | Vertical root padding. |

| `authLayoutPanelPadding` | `--vf-layout-auth-layout-panel-padding` | Panel padding. |

| `authLayoutPanelGap` | `--vf-layout-auth-layout-panel-gap` | Gap between panel sections. |
| `authLayoutPanelBackground` | `--vf-layout-auth-layout-panel-background` | Panel background. |
| `authLayoutPanelBorder` | `--vf-layout-auth-layout-panel-border` | Panel border. |
| `authLayoutHeaderGap` | `--vf-layout-auth-layout-header-gap` | Header internal gap. |
| `authLayoutTitleFontSize` | `--vf-layout-auth-layout-title-font-size` | Title font size. |
| `authLayoutTitleLineHeight` | `--vf-layout-auth-layout-title-line-height` | Title line height. |
| `authLayoutDescriptionColor` | `--vf-layout-auth-layout-description-color` | Description text color. |
| `authLayoutDescriptionLineHeight` | `--vf-layout-auth-layout-description-line-height` | Description line height. |
| `authLayoutFooterColor` | `--vf-layout-auth-layout-footer-color` | Footer text color. |
| `authLayoutFooterLineHeight` | `--vf-layout-auth-layout-footer-line-height` | Footer line height. |
