## Design Tokens

Available design tokens and corresponding CSS variables.

| Token | CSS Variable | Description |
| --- | --- | --- |
| Sticky offsets | `--vf-layout-sticky-subheader-offset-default`, `--vf-layout-sticky-top-offset-with-subheader` | Default sticky subheader offset and combined sticky top offset when a subheader is present. |
| Header sticky z-index | `--vf-layout-document-layout-header-sticky-z-index` | Stacking level for a sticky document header. |
| Subheader sticky z-index | `--vf-layout-document-layout-subheader-sticky-z-index` | Stacking level for a sticky document subheader. |
| Content subheader sticky z-index | `--vf-layout-document-layout-content-subheader-sticky-z-index` | Stacking level for a sticky content subheader. |
| Edge notches | `--vf-layout-document-layout-edge-notch-*` | Color, size, and offset for optional document edge notches. |
| Layout areas | `--vf-layout-header-*`, `--vf-layout-subheader-*`, `--vf-layout-content-*`, `--vf-layout-sidebar-*`, `--vf-layout-aside-*`, `--vf-layout-footer-*` | Area sizing, padding, background, and border tokens used by `VfDocumentLayout`. |
