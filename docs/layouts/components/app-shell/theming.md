## Design Tokens

Available design tokens and corresponding CSS variables.

| Token | CSS Variable | Description |
| --- | --- | --- |
| Sticky offsets | `--vf-layout-sticky-subheader-offset-default`, `--vf-layout-sticky-top-offset-with-subheader` | Default sticky subheader offset and combined sticky top offset when a subheader is present. |
| Header sticky z-index | `--vf-layout-app-shell-header-sticky-z-index` | Stacking level for a sticky app-shell header. |
| Subheader sticky z-index | `--vf-layout-app-shell-subheader-sticky-z-index` | Stacking level for a sticky app-shell subheader. |
| Content subheader sticky z-index | `--vf-layout-app-shell-content-subheader-sticky-z-index` | Stacking level for a sticky content subheader. |
| Layout areas | `--vf-layout-header-*`, `--vf-layout-subheader-*`, `--vf-layout-content-*`, `--vf-layout-sidebar-*`, `--vf-layout-aside-*`, `--vf-layout-footer-*` | Area sizing, padding, background, and border tokens used by `VfAppShell`. |
