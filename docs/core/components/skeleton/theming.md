# Theming

`VfSkeleton` inherits surface palette and radius from shared core tokens:

- `--vf-color-surface-muted`
- `--vf-radius-overlay`

For custom shape/size per-instance, prefer component props (`minHeight`, `radius`) over global CSS overrides.
