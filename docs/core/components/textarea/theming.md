## Design Tokens

Available design tokens and corresponding CSS variables.

| Token | CSS Variable | Description |
| --- | --- | --- |
| Textarea palette | `--vf-textarea-background`, `--vf-textarea-border-color`, `--vf-textarea-color` | Base textarea surface, border, and text colors. |
| Textarea states | `--vf-textarea-hover-border-color`, `--vf-textarea-focus-border-color`, `--vf-textarea-invalid-border-color` | Neutral hover plus semantic focus and invalid border colors. |
| Textarea disabled state | `--vf-textarea-disabled-background`, `--vf-textarea-disabled-color` | Disabled textarea surface and text color. |
| Textarea placeholder | `--vf-textarea-placeholder-color` | Placeholder text color. |
| Textarea min height | `--vf-textarea-min-height-sm`, `--vf-textarea-min-height-md`, `--vf-textarea-min-height-lg` | Minimum textarea height by size. |
| Floating label offsets | `--vf-field-floating-control-offset-inline-*`, `--vf-field-floating-textarea-*` | Inline and resting offsets used when `VfTextarea` is rendered in a floating `VfField`. |
