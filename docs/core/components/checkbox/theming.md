## Design Tokens

Available design tokens and corresponding CSS variables.

| Token | CSS Variable | Description |
| --- | --- | --- |
| Checkbox size | `--vf-checkbox-size-sm`, `--vf-checkbox-size-md`, `--vf-checkbox-size-lg` | Control box size by component size. |
| Checkbox offset | `--vf-checkbox-control-offset-sm`, `--vf-checkbox-control-offset-md`, `--vf-checkbox-control-offset-lg` | Vertical alignment offset by component size. |
| Checkbox palette | `--vf-checkbox-background`, `--vf-checkbox-border-color`, `--vf-checkbox-checked-background`, `--vf-checkbox-mark-color` | Base, border, checked, and mark colors. |
| Checkbox states | `--vf-checkbox-hover-border-color`, `--vf-checkbox-focus-border-color`, `--vf-checkbox-invalid-border-color` | Border colors for hover, focus, and invalid states. |
| Checkbox disabled state | `--vf-checkbox-disabled-*` | Disabled surface, border, text, and mark colors. |
| Checkbox mark geometry | `--vf-checkbox-mark-width`, `--vf-checkbox-mark-height`, `--vf-checkbox-mark-stroke-width`, `--vf-checkbox-mark-offset-y` | Checkmark dimensions and vertical offset. |
