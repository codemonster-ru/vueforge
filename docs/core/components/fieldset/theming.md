# Theming

Component styling surface: available classes and design tokens.

## CSS Classes

Documented CSS classes and their intended usage.

| Class | Description |
| --- | --- |
| `.vf-fieldset` | Root grouped field wrapper with vertical spacing. |
| `.vf-fieldset__legend` | Visible group legend. |
| `.vf-fieldset__description` | Helper text below the grouped controls. |
| `.vf-fieldset__error` | Error text below the grouped controls. |

## Design Tokens

Available design tokens and corresponding CSS variables.

| Token | CSS Variable | Description |
| --- | --- | --- |
| Fieldset legend typography | `--vf-text-label-*` | Legend font size, weight, and line height. |
| Fieldset helper typography | `--vf-text-caption-*` | Description and error text typography. |
| Fieldset spacing | `--vf-surface-gap-tight` | Vertical gap between fieldset parts. |
| Helper text color | `--vf-color-muted` | Neutral helper text tone. |
| Error text color | `--vf-color-danger` | Error text tone. |
