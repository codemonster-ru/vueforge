## Design Tokens

Available design tokens and corresponding CSS variables.

| Token | CSS Variable | Description |
| --- | --- | --- |
| Drawer size | `--vf-drawer-size-sm`, `--vf-drawer-size-md`, `--vf-drawer-size-lg` | Drawer panel size by component size. |
| Drawer surface | `--vf-drawer-content-background`, `--vf-drawer-content-border-color`, `--vf-drawer-content-color` | Panel background, border, and text colors. |
| Drawer focus | `--vf-drawer-content-focus-border-color` | Border color used for focus-visible state. |
| Drawer chrome | `--vf-drawer-header-min-height`, `--vf-drawer-footer-min-height`, `--vf-drawer-divider-color` | Header/footer minimum heights and divider color. |
| Drawer offsets | `--vf-drawer-offset-x-rest`, `--vf-drawer-offset-y-rest`, `--vf-drawer-offset-x-left`, `--vf-drawer-offset-x-right`, `--vf-drawer-offset-y-top`, `--vf-drawer-offset-y-bottom` | Translation offsets used by placement transitions. |
