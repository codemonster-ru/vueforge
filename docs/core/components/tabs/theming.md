## Design Tokens

Available design tokens and corresponding CSS variables.

| Token | CSS Variable | Description |
| --- | --- | --- |
| Tabs list | `--vf-tabs-list-background`, `--vf-tabs-list-border-color`, `--vf-tabs-list-padding-bottom` | List background, border, and baseline padding. |
| Tab item | `--vf-tabs-tab-color`, `--vf-tabs-tab-background`, `--vf-tabs-tab-padding-inline` | Base tab color, background, and horizontal padding. |
| Tab states | `--vf-tabs-tab-hover-*`, `--vf-tabs-tab-active-*`, `--vf-tabs-tab-disabled-color` | Neutral hover and active text plus disabled tab colors. |
| Indicator | `--vf-tabs-indicator-color` | Primary active-state accent. |
| Scroller | `--vf-tabs-scroll-button-width`, `--vf-tabs-fade-width`, `--vf-horizontal-scroller-icon-size-default` | Horizontal scroller button, fade, and icon size defaults. |
| Panel | `--vf-tabs-panel-color`, `--vf-tabs-panel-focus-border-color`, `--vf-tabs-panel-padding-top` | Tab panel color, focus border, and spacing. |
