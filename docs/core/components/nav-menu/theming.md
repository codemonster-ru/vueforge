## Design Tokens

Available design tokens and corresponding CSS variables.

| Token | CSS Variable | Description |
| --- | --- | --- |
| Nav item palette | `--vf-nav-menu-item-color`, `--vf-nav-menu-item-hover-*`, `--vf-nav-menu-item-active-*` | Base, hover, and active item colors. |
| Nav item layout | `--vf-nav-menu-item-padding`, `--vf-nav-menu-item-gap`, `--vf-nav-menu-item-radius` | Item padding, content gap, and radius. |
| Current layout defaults | `--vf-nav-menu-current-list-gap-default`, `--vf-nav-menu-current-nested-gap-default`, `--vf-nav-menu-current-collapse-offset-top-default`, `--vf-nav-menu-current-group-padding-left-default` | Default values used by current variant aliases. |
| Pills variant | `--vf-nav-menu-pills-*`, `--vf-nav-menu-pills-group-padding-left` | Layout and state tokens for the pills variant. |
| Leading icon | `--vf-nav-menu-leading-icon-size`, `--vf-nav-menu-leading-icon-column-size` | Leading icon size and reserved column width. |
| Group label | `--vf-nav-menu-group-label-color`, `--vf-nav-menu-group-font-size`, `--vf-nav-menu-group-padding` | Group label color, typography, and spacing. |
