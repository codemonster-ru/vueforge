## Design Tokens

Available design tokens and corresponding CSS variables.

| Token | CSS Variable | Description |
| --- | --- | --- |
| Switch size | `--vf-switch-width-*`, `--vf-switch-height-*`, `--vf-switch-thumb-size-*` | Track and thumb dimensions by component size. |
| Switch offset | `--vf-switch-control-offset-sm`, `--vf-switch-control-offset-md`, `--vf-switch-control-offset-lg` | Vertical alignment offset by component size. |
| Switch track palette | `--vf-switch-track-*` | Track colors for base, hover, checked, invalid, static, and disabled states. |
| Switch thumb palette | `--vf-switch-thumb-*` | Thumb colors, border colors, shadows, and inverse contrast colors. |
| Switch thumb icon size | `--vf-switch-thumb-icon-size` | Icon size used inside the thumb for icon-based switch variants. |
| Switch thumb icon size defaults | `--vf-switch-thumb-icon-size-sm`, `--vf-switch-thumb-icon-size-lg` | Size-specific icon defaults for small and large switches. |
