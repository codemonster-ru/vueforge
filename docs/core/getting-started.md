# Getting started

Foundation package for VueForge UI: components, composables, breakpoints/foundation helpers, and theme provider/runtime integration for Vue 3 apps.

Core supports both global plugin setup and component-level imports.
For optimized bundles, prefer component subpath imports (for example `@codemonster-ru/vueforge-core/button`) so CSS is linked automatically per component.
