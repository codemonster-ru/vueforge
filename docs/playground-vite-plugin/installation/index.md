# Installation

## Overview

Install the package and import the public entry points.


## Install

```bash
npm i -D @codemonster-ru/vueforge-playground-vite-plugin
```

## Import

```ts
import { vueforgePlaygroundVirtualPlugin } from '@codemonster-ru/vueforge-playground-vite-plugin';
```
