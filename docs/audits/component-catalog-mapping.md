# Component Catalog Mapping

This catalog is maintained only for the current public surface:

- `@codemonster-ru/vueforge`

## Mapping policy

- Track only actively exported components.
- Record alias and overlap decisions only for active components.
- Do not keep removed chart/block experiments in this matrix.

- Core primitives
