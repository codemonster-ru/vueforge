# Component API Package Specs

Current package surface:

- `@codemonster-ru/vueforge`
- `@codemonster-ru/vueforge-layouts`

## Required API review areas

- props
- events
- slots
- accessibility
- responsive behavior
- SSR and hydration behavior
- theme/token coverage
- tests

## Scope rule

- Package specs are maintained only for components that are exported by the active packages.
- Removed or deferred component families should not stay in this document as historical placeholders.
