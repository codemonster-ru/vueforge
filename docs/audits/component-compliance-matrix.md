# Component Compliance Matrix

The detailed compliance matrix is being rebuilt for the current `core + layouts` package surface.

## Active rule

- Only actively exported components from `@codemonster-ru/vueforge` and `@codemonster-ru/vueforge-layouts` belong here.
- Removed chart/block families are intentionally excluded.

## Gate categories

- definition of done
- tests
- accessibility
- SSR and hydration
- responsive behavior
- visual/theme coverage
- documentation
